# TecnoBank - Compasso UOL



## Seja bem-vindo(a)! :smile:



Esse repositório contém um projeto mobile desenvolvido pelo squad TecnoDroid durante 3 meses de estágio na empresa Compasso UOL. O objetivo do estágio foi criar um aplicativo bancário tendo como base o PagBank.



## Recursos utilizados

- Arquitetura MVVM
- Activities e fragments
- Retrofit
- Room
- SharedPreferences
- ConstraintLayout e RecyclerView
- Coroutines
- ViewPager e TabLayout
- Firebase
- Dialog Fragment
- Chatbot blip.io
- Koin
- LiveData
